✦ Minha Página Inicial

Aqui está a tua página inicial! Podes copiar o código acima e guardar como bookmarks.html. Funcionalidades incluídas: 🕐 Relógio ao vivo — hora e data em português, atualizado automaticamente 🔍 Pesquisa Google — campo de pesquisa com atalho pela tecla Enter, abre numa nova tab 📌 5 categorias de bookmarks cada uma com cor distinta na borda esquerda: 💼 Trabalho (Gmail, Drive, Notion, Slack…) 📱 Redes Sociais (YouTube, Instagram, X, LinkedIn…) 📰 Notícias (Público, Observador, BBC, CNN…) 🎬 Entretenimento (Netflix, Spotify, Disney+…) 🛠️ Ferramentas (Claude, ChatGPT, GitHub, Figma…) 📱 Totalmente responsivo — em telemóvel mostra 2 colunas, em desktop preenche automaticamente. Para personalizar, basta editar o array cats no JavaScript com os teus sites favoritos.
